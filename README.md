結婚相談所の会員さんが、仮交際、真剣交際と進めていく中で、話しにくいことを話して成婚へ進めていく仲介役のアプリを開発したい。お互いの生まれ育った文化や習慣から、同じ言葉でも前提や意味するものが違うので、誤解や妄想でうまくいかなくなるのを防ぐ。お互いの言葉の奥にある想いを聞き取って、ずれている会話の翻訳をし、理解を進める。第３案や中韓どころを見つけられるようにする。
使用料については、まずは何人かで使ってみて、あすなる愛媛の結婚相談所の費用負担が大きくなるようであれば、オプション会費をいただくよう検討する。
将来、夫婦関係をよくするアプリとして開発し、スマホにダウンロードできる形でサブスク販売もアリ。
すれ違い、誤解を防いで、言葉の奥の意図を汲み取れるカップルを増やして、世界中に仲良く幸せな家族を増やしていく。
